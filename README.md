# GPU-GPU Communications & Benchmarks (GCB)
GCB develops a suite of apps and kernels to demonstrate, study (and optimize) GPU-GPU communications using new technologies/features in MPI, NCCL, NVSHMEM, and C++.

# Dependencies

* MPI
* NVHPC
* C++
* NCCL
* NVSHMEM
* argparse
* optional: magic_enum
